# 项目简介

## 核心功能
- 基于 YOLOv8 的棉花成熟度检测系统
- 支持 5 类棉花识别：'Cotton Blossom', 'Cotton Bud', 'Early Boll', 'Matured Cotton Boll', 'Split Cotton Boll'
- 提供 4500 张标注数据集
- 支持图片、视频及摄像头进行检测
- 支持批量检测并在界面直接查看所有结果
- 支持检测结果保存
- 提供训练好的模型及训练结果
- 支持自定义置信度和 IOU 阈值
- 可选择显示标签和原图
- 可选择不同的检测模型
- 提供完整的项目运行教程

## 适用场景
- 棉花成熟度检测与分析
- 农业研究中的棉花生长监测
- 棉花生产质量管理
- 棉花采收时机判断

## 技术特点
- 基于 YOLOv8 模型实现
- 使用 PyQt5 构建用户界面
- 支持多种输入源（图片、视频、摄像头）
- 配置好的开发环境：Python3.9、OpenCV-Python、PyQt5、Ultralytics、Torch1.9
- 提供完整的源码和标注数据集
